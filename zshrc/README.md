# Zsh plugins & CLI tools :
## I used arch package manager pacman,you can install them with others like apt,brew or ..
```sh
pacman -S fzf

pacman -S fd

pacman -S eza

pacman -S tldr

pacman -S thefuck

pacman -S zoxide
```
```sh
cp .zshrc ~
``` 
###### or copy the content to your existing .zshrc file
